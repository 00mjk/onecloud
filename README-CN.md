## 什么是 OneCloud?

OneCloud 是一个开源的多云管理平台，具备以下特性:

- 统一全面

  统一API、镜像、调度、账号体系、监控和计费等操作，能够全面管理 On premises、 私有云、公有云资源。

- 内置私有云

  OneCloud 本身就提供完备的私有云实现，提供虚拟机、物理机、容器和负载均衡等资源管理。
